Pomoshnik

A FacebookB chat bot for the Moscow Metro

## 0. Pre-requirements

1. Install [ngrok](https://ngrok.com/).

2. Install nodemon
```
npm install -g nodemon
```
3. Fill the .env file
```
cp .env.example .env
```

## 1. Run the app
Start the tunnel
```
./ngrok http 3000
```
Star the Bot
```
nodemon
```
